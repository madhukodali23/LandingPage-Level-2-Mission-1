# Prompts Used

1. Today I want to build a landing page using HTML, CSS, and JavaScript. What basic concepts should I understand before starting?

2. Can you explain how Flexbox and CSS Grid work with simple examples so I can use them for layout design?

3. What are common mistakes beginners make while building responsive websites and how can I avoid them?

4. How can I implement a dark mode toggle using JavaScript by adding or removing a class on the body?

5. While building a landing page, what kind of bugs or layout issues might occur and how can I debug them?
