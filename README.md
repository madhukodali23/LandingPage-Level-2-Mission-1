# Prodesk IT Digital Agency - Landing Page

This is a responsive landing page built as part of Week 1 internship mission at Prodesk IT.

The project focuses on core frontend fundamentals including layout design, responsive behavior, and modern CSS styling.

## Features

* Responsive design (Mobile + Desktop)
* Sticky navigation bar
* Dark mode toggle
* Modern hero section with call-to-action
* Services section with interactive cards
* About section with structured layout
* Contact section with clean UI
* Hover animations and micro-interactions

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (Vanilla JS)
